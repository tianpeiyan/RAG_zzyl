**联调顺序清单**
这部分最关键，建议严格按顺序做。

1. 联调 L1：Python 自测完成后再开放给 Java

   通过标准：

   - Python 四个内部接口全部可用
   - 30 条问题首轮正确率达到 >= 70%

2. 联调 L2：Java 单独联通 Python

   通过标准：

   - Java 调用 Python 成功率 >= 95%

3. 联调 L3：前端接入 Java 聊天接口

   通过标准：

   - 前端不改代理配置也能完成提问
   - 提问到回答完整链路打通

   通过标准：

   - 前端点击“重建索引”后，Java 能转发到 Python
   - Python 重建状态可回显到前端

   通过标准：

   - 30 条问题整体正确率 >= 80%
